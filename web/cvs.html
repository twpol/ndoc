NDoc CVS Instructions

If you like living on the bleeding edge, feel free to check out the latest code as often as you like.

  1. Create a directory to hold the project. (I use C:\SourceForge\ but you can use whatever you want.)
  2. Download CVS for Windows and place cvs.exe in your new directory. (Be careful that you don't extract it to the cvsM-NN sub-directory or the batch file won't work.)
  3. Download the checkout batch file and place it in the same directory.
  4. Double-click the checkout.bat file and hit ENTER when prompted for your CVS password.

That's it!
