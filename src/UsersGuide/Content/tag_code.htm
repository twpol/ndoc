The <code> tag gives you a way to indicate multiple lines as code.

<code>content</code>

where:

content
The text you want marked as code.

Applies To

Can be used inline within any other markup.

Remarks

Use <c> to indicate that text within a description should be marked as code.

Example

See the <example> topic for an example of how to use the <code> tag.
