Overview

The XML Documenter is the simplest of the NDoc Documenters. It is primarily a development tool. 

As part of the documentation compile process, NDoc merges the type information in the assemblies being documented with the code comment summary XML document that the /doc compiler option emits. The XML Documenter allows you to save this merged set of data for curiosity or debugging purposes. Used in conjunction with the UseNDocXmlFile setting, this is especially useful when you are working on your own documenters.
